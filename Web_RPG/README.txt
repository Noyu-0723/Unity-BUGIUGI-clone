/* 制作物について */
概要: 
Web上で動作するRPGゲームです。
基本的には、画面上に表示されているボタンをクリックして進めていく感じになるかと思います。
戦闘システム: 
プレイヤーと敵が交互に攻撃し合うターン制バトルです。
プレイヤーは通常攻撃の他、回復魔法・攻撃魔法・ポーションを使うことが出来ます。戦況に応じて上手く立ち回りましょう。
また、勝てないとは判断した場合には逃げることもできます。時にはそれが最善手となるかもしれません。
街について: 
スタート地点である街では、「武器・防具の購入」と「酒場での休憩」をすることが出来ます。
武器・防具は、購入すると自動的に装備されます。
酒場で休憩した場合、HP/MPの回復・ポーションの補充をすることが出来ます。

/* 使用言語 */
HTML/css
JavaScript

/* プレイ方法 */
1. 任意の場所にファイルを保存する
2. エクスプローラー上で「index.html」を保存した場所まで移動する
3. 「index.html」をダブルクリックしてWeb上で実行する(※Google Chrome推奨)
