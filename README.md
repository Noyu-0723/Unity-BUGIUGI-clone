## インストール
各ゲームをローカルで実行するには、以下の手順に従ってください：

1. リポジトリをクローンします：
    ```bash
    git clone https://github.com/Noyu-0723/my-portfolio.git
    cd my-portfolio
    ```

2. 実行したいゲームのディレクトリに移動します：
    ```bash
    cd game-directory
    ```

3. 各ゲームのREADMEファイルに記載されている特定の手順に従ってください。  
※ゲームによってはコンソール上からの実行ではないかもしれません。
